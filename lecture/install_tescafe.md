# Chapter 2 - Install NodeJS , VS Code and TestCafe on macOS

#### Requisites
- NodeJS and NPM
- Visual Studio Code
- Install the recommended plugins for Visual Studio and TestCafe:
  - TestCafe Snippets
  - TestCafe Test Runner
  - TestLatte
